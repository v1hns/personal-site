Minecraft AI Bots

Autonomous AI agents that play Minecraft — navigating, crafting, and surviving in a complex open-world environment through reinforcement learning and behavioral programming.

GitHub →

Overview

Minecraft is a uniquely challenging environment for AI agents — it's open-ended, partially observable, and requires long-horizon planning across thousands of sequential decisions. Building bots that can survive and thrive in this world requires combining low-level behavioral programming with higher-level goal planning.

This project develops autonomous Minecraft agents capable of resource gathering, crafting, navigation, and survival — serving as a platform for experimenting with agent architectures, reinforcement learning, and LLM-based task planning in complex environments.

Technical approach

The bots combine rule-based behavioral trees for low-level actions with ML-driven planning for goal decomposition. Experiments include:

  • Pathfinding and navigation in procedurally generated terrain
  • Resource gathering and inventory management via state machines
  • LLM-driven task decomposition for high-level goal completion
  • Multi-agent coordination experiments
