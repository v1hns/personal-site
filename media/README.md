# Media

Assets for vihaanshringi.com

## Structure

```
media/
├── photos/    — profile photo, event photos, lab photos, etc.
└── videos/    — vlog clips, demo videos, etc.
```

## Usage

Reference files in `index.html` like:
```html
<img src="media/photos/profile.jpg" alt="Vihaan Shringi">
```

Recommended formats: `.jpg` / `.webp` for photos, `.mp4` / `.webm` for videos.
